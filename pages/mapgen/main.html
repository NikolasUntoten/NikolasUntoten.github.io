This is a map making utility, currently using a version of Perlin noise, and a lot of spice to make it look like a map.

Controls: Generate draws a low-resolution version of the picture, and redraw draws a higher resolution version (based on your selected quality). Classic mode gives the map a nice old-timey feel, and without it there is a modern green/red topography color spectrum.

This UI is super basic, and only for testing really. Please don't try to break it, it's fragile ;~;
